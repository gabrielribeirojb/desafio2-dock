
# Como executar o projeto

## Back end
Pré-requisitos: Java 8

```bash
# clonar repositório
git clone https://github.com/gabrielribeirojb/desafio2-dock

# entrar na pasta do projeto back end
cd Desafio-API-Rest-Java

# executar o projeto
./mvnw spring-boot:run
```
